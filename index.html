Основные виды селекторов

  • * - Выбирает все элементы на странице.
  • Теги - Выбирает все элементы с указаным тегом.
  • #id - Выбирает элемент с указаным id.
  • .class - Выбирает все элемнты с указаным классом.
  • [name="value"] - Выбирает элементы с определенным атрибутом, его значени и условиях
  • :visited - "Псевдоклассы" Псевдо-класс – это простой селектор, который используется для выбора того, что не может быть выбрано с помощью других селекторов или может быть выбрано, но достаточно трудным способом.
